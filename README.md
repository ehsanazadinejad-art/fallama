# 🦙 Fallama (فالاما) | مترجم هوشمند و آفلاین زیرنویس

![Fallama Banner](https://via.placeholder.com/800x400?text=Fallama+-+AI+Subtitle+Translator)
<img width="936" height="597" alt="fallama screenshot" src="https://github.com/user-attachments/assets/2ce59b4f-4eb4-4ae0-8b82-fdbec1cc8e75" />
**فالاما**، ترکیب قدرت هوش مصنوعی (Ollama) و زبان شیرین فارسی (Fa) است! 
یک نرم‌افزار کاملاً رایگان، متن‌باز و **۱۰۰٪ آفلاین** که فایل‌های زیرنویس (SRT) شما را با بالاترین کیفیت و بدون نیاز به اینترنت ترجمه می‌کند. 

---

## ❓ فالاما دقیقاً چیست و چه کار می‌کند؟
ترجمه فایل‌های زیرنویس همیشه دردسرهای خودش را داشته؛ از به هم ریختن زمان‌بندی‌ها (Timecodes) گرفته تا ترجمه‌های ماشینی و بی‌روحِ گوگل ترنسلیت. 

فالاما این مشکل را حل کرده است. شما فایل‌های `.srt` خود را به برنامه می‌دهید، لحن ترجمه (رسمی، عامیانه یا کلمه‌ای) را انتخاب می‌کنید، و فالاما با استفاده از مدل‌های زبانی قدرتمند (مثل `translategemma`) روی گرافیک و پردازنده سیستم **خودتان**، یک ترجمه روان، دقیق و آماده‌ی تماشا تحویل می‌دهد.

---

## 🎯 فالاما به درد چه کسانی می‌خورد؟

🎬 **مترجمین و تیم‌های دوبله:**
اگر می‌خواهید یک ترجمه اولیه (Draft) بسیار باکیفیت و با لحن «شکسته و عامیانه» داشته باشید تا سرعت کارتان ۱۰ برابر شود، فالاما دستیار شماست.

🍿 **عشاق فیلم و سریال (Movie Geeks):**
فیلمی دانلود کرده‌اید که هنوز زیرنویس فارسی ندارد؟ کافیست زیرنویس انگلیسی را به فالاما بدهید و چند دقیقه بعد، فیلم را با زیرنویس فارسی تماشا کنید.

🔒 **افراد حساس به حریم خصوصی:**
از آنجایی که هیچ دیتایی به هیچ سروری در اینترنت ارسال نمی‌شود، فالاما برای ترجمه اسناد و ویدیوهای محرمانه یا شرکتی، امن‌ترین گزینه ممکن است.

---

## ✨ ویژگی‌های قاتل (Killer Features)
- **🛡️ ۱۰۰٪ آفلاین و لوکال:** بدون نیاز به API، بدون نیاز به اینترنت، بدون تحریم!
- **🎨 رابط کاربری مدرن:** یک محیط تاریک، شیشه‌ای و چشم‌نواز با انیمیشن‌های روانِ الاستیک که کار با آن لذت‌بخش است.
- **⏱️ مانیتورینگ زنده سخت‌افزار:** نمایش لحظه‌ای میزان درگیری CPU, RAM و GPU سیستم شما در حین ترجمه.
- **🎭 لحن‌های ترجمه هوشمند:** قابلیت انتخاب بین لحن‌های رسمی (کتابی)، شکسته (مخصوص دوبله و فیلم) و ترجمه دقیق.
- **📂 پردازش دسته‌ای (Batch):** می‌توانید ۱۰ فایل زیرنویس را با هم به برنامه بدهید و در نهایت یک فایل ZIPِ تر و تمیز تحویل بگیرید.
- **🛠️ حفظ ساختار SRT:** فالاما به هیچ‌وجه زمان‌بندی‌ها و ساختار خطوط زیرنویس را خراب نمی‌کند.

## 🚀 How to Use (راهنمای اجرا)

### روش اول: برای کاربران عادی (نصب سریع)
نیازی به برنامه‌نویسی نیست! 
1. نرم‌افزار هوش مصنوعی [Ollama](https://ollama.com/) را نصب کنید.
2. به بخش **[Releases](../../releases)** بروید و فایل `Fallama_Setup.exe` را دانلود و نصب کنید.
3. برنامه را باز کنید و لذت ببرید!

### روش دوم: برای توسعه‌دهندگان (اجرای سورس‌کد)
```bash
git clone [https://github.com/ehsanazadinejad-art/fallama.git](https://github.com/ehsanazadinejad-art/fallama.git)
cd fallama
pip install -r requirements.txt
python main.py
